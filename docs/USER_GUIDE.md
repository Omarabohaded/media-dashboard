# Dashboard User Workflow

## Add and prepare a client

1. Open Admin and create the client with website platform and reporting currency.
2. Select the client in the shared client selector.
3. Connect the storefront when available; otherwise mark store access declined.
4. Connect each paid platform and save exactly one advertiser/customer/ad account to this client.
5. Discover conversion events, then save exactly one purchases event and one purchase-value event. Use a global default only when it is genuinely correct for all clients; otherwise save a client override.
6. Open Health and resolve missing connection, account, mapping, expired-token, failed-sync, or stale-data states.
7. Open Paid Media for the single-client channel/blended view.
8. Enable owner mode and open Portfolio for cross-client reporting. Do not compare raw money totals across different currencies until an FX layer exists.

Deleting a client removes its client-scoped Meta, storefront, TikTok, Google Ads, and Snapchat connection records. At least one client must remain.
